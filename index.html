Github repos

Github profile: KateMMiller
forestMIDN: R package for summarizing and compiling MIDN forest data
forestNETN: R package for summarizing and compiling NETN forest data
forestQAQC: QA/QC R markdown reports for weekly checks, QA/QC reports, and compiling previous visit data forms.
NETN_water_summaries: R Markdown reports by park to summarize monthly water data relative to historic data.
wetlandACAD: R package for summarizing and compiling freshwater wetland data in Acadia National Park
wetlandViz: R shiny app to visualize freshwater wetland data in Acadia National Park
forestTrends: R package for non-parametric trend analysis with mixed models. Package includes non-parametric power simulations.
rockyIntertidal: R package for rocky intertidal data, including compiling temp. logger data relative to high tide and compiling, summarizing, and visualizing community data.
